# Luke Daniel - Personal Portfolio

Welcome to the repository for my personal portfolio website!

🌍 **Live Site:** [https://LukeBDaniel.github.io](https://LukeBDaniel.github.io)

This website is built using [Astro](https://astro.build/) and serves as a central hub for my projects, resume, and blog posts. It features a sleek, minimalist design with interactive glassmorphism elements.

## 🚀 Projects Featured

- **AMC Guide:** A web application with a custom scraper and React frontend for filtering and organizing AMC movie showtimes.
- **GASP Air Quality Dashboard:** A data analytics capstone project visualizing 5 years of air quality violations and fines from major industrial polluters.
- **NY Sports Attendance Predictor:** An interactive Machine Learning tool predicting home game attendance for major NY-area sports teams.

## 🛠️ Local Development

If you'd like to run this project locally:

1. Clone the repository:
   ```sh
   git clone https://github.com/LukeBDaniel/LukeBDaniel.github.io.git
   ```
2. Navigate into the directory:
   ```sh
   cd LukeBDaniel.github.io
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the local development server:
   ```sh
   npm run dev
   ```
5. Open `localhost:4321` in your browser.
